## 快速开始

下面给出本地搭建项目的示例步骤。

### 前置条件

列出运行软件所需的依赖与安装方式。
* npm
  ```sh
  npm install npm@latest -g
  ```

### 安装

1. 在 [https://example.com](https://example.com) 申请 API Key
2. 克隆仓库
   ```sh
   git clone https://github.com/OWNER/REPO.git
   ```
3. 安装依赖
   ```sh
   npm install
   ```
4. 把 API Key 写入 `config.js`
   ```js
   const API_KEY = 'ENTER YOUR API';
   ```
